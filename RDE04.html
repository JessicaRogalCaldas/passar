HTML

É uma Linguagem de Marcação de Hipertexto baseada em tags criada na década de 1990 por Tim Berners-Lee.

A HTML é utilizada para publicação de conteúdo en hipertexto, tais como, texto, imagem áudio e video , e que podem ser vinculados através de hyperlinks.

TAGS

As Tags definem ao navegador a marcação de conteúdo.
Tem funcionalidades específicas.
Sem o uso de tags o navegador exibe um texto plano.


As tags podem possuir atributos, ou seja, são parâmetros que definem algum tipo de informação extra ao navegador.

IMAGEM

Representação visual de uma pessoa, objeto, paisagem.

IMAGEM DIGITAL:

Representação de uma imagem bidimensional usando números binários codificados de modo a permitir seu armazenamento, transferência, impressão ou reprodução, e processamento por meios eletrônicos.

FORMATOS DE IMAGEM:

Formatos de imagem são padrões de codificação, compressão e definição que determinam o tipo de arquivo gráfico, tais como: BMP, TIFF, GIF, .SVG, .ICO, JPEG, PNG dentre outros.

MAIS UTILIZADOS:
JPEG: Joint,Photographic,Experts,Group.
Arquivo criado na década de 80.
As diferenças entre as extensões de arquivo é que eles vão compactar o arquivo deixando maior ou menor em número de bytes megabytes ou kbytes.
O JPEG preserva o fundo. Em 1996 o W3C inventou outro tipo de formato que é o png (Portable Network Graphics), com o fundo sendo transparente.

MANIPULAÇÃO DE IMAGEM:

Para manipular as imagens, pode-se fazer o uso de de dois softwares ou Gimp ou Photoshop, além de outros, mas esses são os dois principais.
Gimp é livre, pode ser utilizado para edição de imagens e o Photoshop é um software proprietário.
OBS:na hora de manipular imagens tem que cuidar com o tamanho de megabytes ou kilobytes, pois da diferença na hora de carregar o site, deixando mais lento ou não.

MARCAÇÃO DE IMAGENS

As tags de imagens e seus parâmetros definem as imagens e suas propriedades dentro do documento HTML.

  • Tag Img - Vai definir que é uma Imagem.
  • Src - Vai definir de onde está vindo a imagem, já a mostrando.
  • Width - Vai definir a largura em pixels ou percentual da imagem.
  • Heigth - Vai definir a altura em pixels ou percentual da imagem.
  • Alt - Vai definir a descrição. Vai fazer uma leitura da descrição da imagem para a pessoa.
  • Figure - tag semânticas para uma organização e significação do documento, encontrando a imagem.
  • Figcaption - Vai mostrar uma legenda logo abaixo da imagem.

OBS: É necessário que largura e altura seja próxima ao que vai ser exibido em tela para não demorar para carregar, por que vai reduzir o tamanho do arquivo em MB

Figure e Figcaption foram definidas a partir do HMTL 5
