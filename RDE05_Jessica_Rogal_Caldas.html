HIPERLINK OU LINK

É uma referência dentro de um documento em hipertexto para outras partes do mesmo documento, outros documentos, arquivos ou endereço eletrônico.
Os links podem ser Remotos ou externos, Internos, Locais e com Endereço de correio ou arquivos.
Além de hiperlinks em imagens

MARCAÇÃO DE HIPERLINKS

A marcação de hiperlinks acontece utilizando a tag < a > e a propriedade href que faz a referência para o conteúdo desejado.

HIPERLINKS EXTERNOS OU REMOTOS

São links que referenciam sites ou páginas da web. Na propriedade href utilizamos a URL (Uniform Resource Locator) para a referência que desejamos.
Lembrando que a URL(Uniform Resource Locator) é o endereço.

  • Href - localiza a referência;
  • Terget="_blank" - bre uma nova guia;
  • Target="_self" - abre na mesma aba;
  • Rel - mostra ao que é referente o link externo, ligando a outra parte.
  • Hreflang="en" - define qual é a linguagem, ativando a opção para tradução automática

HIPERLINKS INTERNOS

São links que apontam a um lugar diferente dentro da mesma página e possibilita acessar diferentes parágrafos ou seções quando o documento é extenso.

HIPERLINKS LOCAIS

São links que apontam para outras páginas dentro de um mesmo site.

HIPERLINKS EM IMAGENS

É possível realizar hiperlinks imagens envolvendo a tag e imagem com a tag de hiperlink.

HIPERLINKS PARA DOWNLOAD DE ARQUIVOS

São links que apontam para arquivos e possibilitam realizar download deste arquivos.

HIPERLINS PARA ENDEREÇO eletrônico

São links que apontam para um endereço eletrônico.
