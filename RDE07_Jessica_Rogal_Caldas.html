BRANCHES

São ramificações de um projeto de software ou documento e tem como objetivo o desenvolvimento paralelo de:

  • Novas funcionalidades
  • Correção de problemas
  • Alteração de código
  • Branche Master ou Main
  • Branche Secundária

    Ter uma versão estável na branch master(principal) que os usuários estão usando, enquanto isso a equipe pode fazer alterações no repositório em branchs secundárias sem prejudicar a principal.

    HOSPEDAGEM

    A etapa de hospedagem de um site é importante por QUEBRA consolida e finaliza seu trabalho e desenvolvimento, onde qualquer suário que tem acesso a nternet poderá ver o conteúdo do site.
    Para hospedar, em geral precisa pagar para hospedar o site, mas o github oferece uma hospedagem gratuita:

    GitHub Pages

    Hospedagem gratuita de sites. Basta criar uma conta e um repositório público e a URL será: https://nomeusuario.github.io/repositorio

    COMO HOSPEDAR NO GITHUB PAGES:

    Abra o github.com e clique em um repositório desejado para fazer a hospedagem no github pages. Depois clique nas configurações.

    Pode realizar diversas alterações.
    Role a página até encontrar o github pages- páginas do github:

    Importante que o repositório esteja público, isso será mostrado na zona de perigo.

    Selecione a branch desejada.

    Salve:

    No momento em que é salvo, automaticamente avisa que o site foi publicado no teu endereço da web, assim o arquivo ja esta hopsedado no site.
    Importante testar uma alteração na branch secundária para ver se realmente não muda na principal.
