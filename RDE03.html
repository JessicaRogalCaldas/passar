PÁGINA WEB E WEBSITE

PÁGINA WEB OU WEBPAGE:

Arquivo que possibilta inserir textos, imagens, tabelas e incorpar planilhas, apresentações, vídeos com ligações de hipertexto que permitem a navegação de uma página, ou seção, para outra.

WEBSITES OU SITES:

Coleção de páginas web armazenadas em uma pasta ou em subpastas relacionadas de um servidor web.

ARQUITETURA CLIENTE - SERVIDOR

O cliente faz uma requisição ao servidor, um processamento ao lado do servidor, no Server-Side retornando para o cliente.
No SERVER - SIDE é preciso ter tecnologias que sejam capazes de atender o cliente, podendo ser as:

  1. Linguagens de Programação;
  2. Banco de Dados;

Além de outras tecnologias necessárias.
Precisa de desenvolvedores ou profissionais que entendam das tecnologias de Server-Side

Feito o processamento, o servidor retorna para o CLIENT - SIDE que também possui tecnologias para o cliente, como:

  1. HTML
  2. CSS
  3. JAVA SCRIPT

Além de outras opcções.

HTML - Hypertext Markup Language

É uma Linguagem de Marcação de Hipertexto baseada em tags criada na década de 1990 por Tim Berners-Lee para publicação de conteúdo em hipertexto, tais como, texto, imagem, áudio e vídeo e que podem ser vincuados através de hyperlinks.

TAGS

As Tags definem ao navegador a marcação de conteúdo.
Tem funcionalidades específicas.
Sem o uso de tags o navegador exibe um texto plano.


As tags podem possuir atributos, ou seja, são parâmetros que definem algum tipo de informação extra ao navegador.

ESTRUTURA BÁSICA DO DOCUMENTO HTML

Definido pela tag HTML.

Dentro das Tags Html, podem existir:

TAG RED
Define cabeçálhio.Dentro dela existe a TAG TITLE
TAG TITLE
Define barra de título para navegador.
TAG BODY
Onde terá todo o conteúdo digitado. Para adicionar comentários, usa-se o parênteses angulado, sinal de exclamação e dois traços, depois do comentário dois traços e fecha-se o parênteses angulado.
DOCTYPE html
Declaração de tipo de documento.Dentro da tag HEAD, indroduz-se as tags META
TAG META
Definem por exemplo, o tipo de tela utilizada
