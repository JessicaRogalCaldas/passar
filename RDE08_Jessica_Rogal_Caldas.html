Table

Tag utilizada para inserir uma tabela HTML.
A tag < table > pode conter um elemento ou mais, como < tr > , < th> e < td > .

  • < tr >: Define uma linha da tabela.
  • < th >: Define cabeçalho.
  • < td >: Define uma célula da tabela.

Uma tabela HTML também pode incluir os elementos < caption > , < colgroup > , < thead > , < tfoot > e < tbody > .

Blockquote, Q

  • Tag < blockquote > especifica uma seção citada de outra fonte e indica que o texto incluído é uma longa citação. Normalmente, este é processado visualmente pelo recuo.
  • Tag < q > define uma pequena citação, implementado cercando o texto entre aspas. Próprio quando não há quebra de linha.

Bdo

A tag < bdo > substitui a direção do texto atual.

  • Dir: direção de texto.
  • ltr: indica que o texto ficará da esquerda para direita ( L eft- t o- R ight).
    • Este texto ficará da esquerda para a direita
  • rtl: indica que o texto ficará da direita para a esquerda ( R eft- T o- L ight).
    • Este texto ficará da direita para a esquerda.

Code

A tag < code > é usada para definir um pedaço de código de computador. Indica que o texto é um pequeno fragmento de código, com o conteúdo interno exibido na fonte monoespaçada padrão do navegador.

Ao citar uma tag por exemplo,strong que é um fragmento de código.

Pre

A tag pre define o texto pré-formatado.
Um texto dentro desse mesmo elemento é aplicável em uma fonte não proporcional da maneira em que o texto original foi estabelecido no arquivo. Espaços em branco são coletivos no texto da mesma forma em que este foi digitado.

O texto se encontrará escrito dessa forma.

Kbd

A tag < kbd > é usada para definir a entrada do teclado. Representa uma extensão de texto embutido denotando a entrada textual do usuário a partir de um teclado, entrada de voz ou qualquer outro dispositivo de entrada de texto.
O conteúdo interno é exibido na fonte monoespaçada padrão do navegador.

Resultado:

Precione Ctrl + C e copie o texto
