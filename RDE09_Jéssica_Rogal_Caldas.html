TAG VÍDEO

É utilizada para incorporar um conteúdo de vídeo no documento html.Útil para que os vídeos colocados sejam do próprio servidor ou máquina local.

  • A partir do momento em que der o play o vídeo irá carregar, antes aparecerá somente a imagem estática.
  • O html 5 dá a possibilidade de carregar outros vídeos caso o primeiro falhe ou o navegador não tenha a compatibilidade, por isso usa-se mais source.
  • Para os controles de vídeo é preciso colocar o controls na tag.

TAG ÁUDIO

Esta tag serve para reproduzirmos uma mídia de áudio.
Como exemplo, através do YouTube, pela sua Biblioteca de Áudio é possível selecionar os áudios.

  • preload="metadata" - irá carregar os metadados do arquivo, ou seja, dados internos do arquivo, como tempo de duração da música.
  • controls - todos os controles, como dar o play, pausar, etc.

TAG PICTURE

Permite mais flexibilidade na especificação de recursos de imagem, assim várias imagens podem ser projetadas para preencher melhor a janela de visualização do navegador.

Uma alternativa para que a imagem mude conforme o espaço, ou seja, se adeque ao espaço é utilizar um container. Colocando o source media= ao tamanho máximo que a tela deve atingir para realizar o redirencionamento da tela.
O srcset que vai exercer a função de chamar uma imagem alternativa, um pouco menor.
